## ğŸ“‹ About EventHub

EventHub is a modern web platform designed for RMIT University's campus community to discover, manage, and participate in student events. The system enables:

- **Students** to browse events, RSVP, and provide feedback
- **Organizers** to create events, track attendance, upload galleries, and gather insights
- **Administrators** to moderate content, manage users, and access analytics

**Key Features:** Event management, smart recommendations, RSVP tracking, calendar view, photo galleries, admin dashboard, and public REST API.
